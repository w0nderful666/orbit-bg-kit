FAQ
Common Questions
Does it need a backend?
No. Orbit Background Kit is 100% frontend. There is no server, no API, no database. Just a script tag.
Does it upload any data?
No. Zero analytics, zero tracking, zero fingerprinting. Nothing leaves the browser.
Can I use it in commercial projects?
Yes. It's MIT licensed. Use it anywhere — personal, commercial, whatever. No attribution required.
How do I deploy to GitHub Pages?
Push the repo to GitHub → Settings → Pages → Deploy from branch → main / root. Your site will be live at https://w0nderful666.github.io/orbit-bg-kit/
How do I pin a specific version on jsDelivr?
Use @v0.1.0 instead of @latest: https://cdn.jsdelivr.net/gh/w0nderful666/orbit-bg-kit@v0.1.0/dist/orbit-bg.iife.js
Why is the background not showing?
Make sure the script loaded (check Network tab). Make sure the <orbit-bg> tag is in the body. If using position="absolute", the parent must have position: relative and a defined height.
How do I control z-index?
Use z-index="0" to put it above the page background but below content. Default is -1 (behind everything). Your content needs position: relative; z-index: 1 or higher to appear above it.
Does it affect page performance?
The component uses CSS transforms and opacity for animation (GPU-accelerated). It pauses when the tab is hidden. It respects prefers-reduced-motion. On most devices it's negligible.
