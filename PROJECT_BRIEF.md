# Project Brief

## Name

Orbit Background Kit

## Repository

`w0nderful666/orbit-bg-kit`

## Version

`0.1.0`

## Goal

Provide a lightweight, original, pure frontend Web Component that lets users add animated backgrounds to any static webpage with one CDN script and one `<orbit-bg>` tag.

## Constraints

- No backend.
- No tracking.
- No uploads.
- No runtime dependencies.
- GitHub Pages friendly.
- jsDelivr CDN friendly.
- Safe to embed in normal pages.

## Primary Users

- Indie makers building landing pages.
- Developers building docs or portfolio sites.
- Static site owners who want a polished background without a framework.
