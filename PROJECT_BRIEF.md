# Orbit Background Kit

> 一行 script + 一个自定义标签，给任意网页添加高级动态背景。

## 一句话介绍
纯前端 Web Component 背景动效库，零依赖、零后端、零追踪。10 个主题、12 个预设、20 个可配置属性。

## 核心卖点
- **一行引入** — 复制两行代码即可使用
- **零依赖** — 不引入任何框架和库
- **Web Component** — Shadow DOM 隔离，不影响用户页面
- **10 个主题** — cream, dark, cyber, paper, terminal, glass, midnight, sunset, matrix, aurora
- **12 个预设** — 一键切换风格
- **20 个属性** — 完全可配置
- **性能友好** — GPU 加速、标签页暂停、减少动画支持
- **隐私安全** — 无后端、无追踪、无 Cookie

## 目标用户
- 前端开发者：需要快速给页面加高级背景
- 设计师：需要展示作品集
- 开源项目：需要好看的项目主页
- AI/SaaS 产品：需要科技感 Landing Page

## 技术栈
- 纯 HTML / CSS / JavaScript
- Web Components (Custom Elements + Shadow DOM)
- 无构建工具依赖
- 适合 GitHub Pages + jsDelivr CDN
