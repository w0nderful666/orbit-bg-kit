# Contributing

Thanks for helping improve Orbit Background Kit.

## Project Principles

- Keep it pure frontend.
- Keep it dependency-free at runtime.
- Do not add analytics, tracking, uploads, or backend requirements.
- Preserve the native Web Component API.
- Keep examples easy to copy.

## Local Workflow

```bash
npm run dev
npm run build
npm run check
```

## Pull Requests

Please include:

- A short description of the change.
- Any affected attributes, themes, presets, or examples.
- Manual testing notes.

Avoid unrelated refactors in the same pull request.
